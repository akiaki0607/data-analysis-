# 多天对比数据可视化报告项目

## 项目概述
本项目专注于多天对比数据可视化报告的生成和分析，主要包含GEO诊断报告、优贝公司数据分析、红旗公司对外报表等内容。

## 项目结构
```
多天对比数据可视化报告/
├── input/                    # 输入数据目录
│   ├── 优贝/                 # 优贝公司数据
│   │   ├── 20251014优贝对外报表 (3).md
│   │   └── 20251022优贝对外报表(1).md
│   └── 其他公司/             # 其他公司数据
│       └── 20251024红旗对外报表.md
├── output/                   # 输出报告目录
│   ├── 优贝/                 # 优贝公司报告
│   │   ├── GEO诊断报告_优贝_20251023.html
│   │   └── 优贝GEO诊断报告-优化前后对比分析-完整版.html
│   └── 红旗/                 # 红旗公司报告
│       └── GEO诊断报告_红旗_20251024.html
└── 需求/                     # 需求文档目录
    ├── GEO诊断报告生成提示词(1).md
    ├── 思迈特GEO引擎诊断报告 - 移山科技GEO策略中心.html
    └── 思迈特报表.md
```

## 使用说明

### 数据输入
将需要分析的数据文件放置在`input/`目录下，按公司或项目分类存放。

### 报告生成
生成的报告文件会保存在`output/`目录下，支持HTML格式的可视化报告。

### 需求文档
项目需求和相关提示词文档存放在`需求/`目录下，便于参考和维护。

## 技术特点
- 支持多天数据对比分析
- 生成可视化HTML报告
- 模块化的项目结构
- 完整的版本控制记录
