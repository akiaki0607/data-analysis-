# 首屏优化测试用例

## 测试目标
验证首屏优化后的功能完整性和性能指标，确保达到以下目标：
- 首屏表单提交率 ≥ 3-4%
- 资源点击-到-提交转化 ≥ 25%
- LCP p75 ≤ 2.5-3.0s

## 测试执行计划

### 阶段1：功能测试（开发完成后立即执行）
- 执行TC001-TC015
- 验证所有功能正常工作

### 阶段2：响应式和兼容性测试（功能测试通过后）
- 执行TC016-TC031
- 确保多端兼容性

### 阶段3：性能测试（上线前）
- 执行TC021-TC025
- 确保性能指标达标

### 阶段4：转化率测试（上线后7-14天）
- 执行TC026-TC027
- 验证业务目标达成
