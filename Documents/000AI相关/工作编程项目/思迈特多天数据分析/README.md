# 思迈特多天数据分析项目

## 项目概述
本项目旨在对思迈特品牌在AI问答平台上的数据表现进行多天对比分析，生成可视化报表，帮助产品经理理解品牌竞争态势和优化方向。

## 主要功能

### 1. 数据可视化报表
- **整体表现对比**：可见概率、推荐概率、信源平台占比、信源文章占比
- **分平台分析**：元宝、DeepSeek、豆包、Kimi四大AI平台
- **趋势分析**：10月5日 vs 10月19日的数据变化
- **薄弱关键词分析**：识别需要优化的关键词组合

### 2. 交互功能
- **Tab切换**：支持不同指标和平台的切换查看
- **数据筛选**：支持按平台和状态筛选关键词
- **响应式设计**：适配不同屏幕尺寸

### 3. 核心发现
- 思迈特整体表现提升显著（可见概率从41.79%提升至56.02%）
- 元宝平台表现突出（可见概率达75%）
- 部分关键词在豆包、Kimi平台上表现薄弱
- 瓴羊在Kimi平台增长明显，需要关注

## 技术栈
- **前端框架**：原生HTML5 + CSS3 + JavaScript
- **图表库**：ECharts 5.4.3
- **样式设计**：响应式布局，现代化UI设计

## 使用方法
1. 直接打开 `思迈特多天数据分析报告_优化版.html` 文件
2. 使用Tab切换查看不同维度的数据
3. 使用筛选器查看特定平台或状态的关键词
4. 图表支持交互操作（缩放、悬停等）

## 数据说明
- **数据周期**：2025年10月5日至10月19日
- **分析品牌**：思迈特（客户）vs 核心竞品（帆软、瓴羊等）
